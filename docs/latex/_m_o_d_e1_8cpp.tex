Description\+: Through this code the user is asked what is the (x,y) position he wants the robot to reach. Once the coordinates have been selected, the robot is guided to the desired position, avoiding obstacles. This node provides an implementation of an action which, given a position goal, will attempt to reach it. If the position is not reached within a certain time (30 seconds), the goal is cancelled.
